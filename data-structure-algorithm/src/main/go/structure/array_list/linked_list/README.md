### 不带头单链表
- 插入:
    - 需要遍历到最后一个插入
- 删除:
    -   头部删除
    -   删除某个下标的节点
    -   删除某个元素
    
    
### 循环单链表
- 定义: 最后一个指针不指向null,指向头节点而已

### 双向链表
- 定义: 每个节点,2个指针域和1个数据域 ,指针域分别指向前驱节点和后继节点
- ![数据结构如图](双向链表.jpg)
- 插入:
    -   链表的插入,一概遵循先连接后断的原则,既需要`先将该节点指向插入节点`,再将前驱节点指向新的节点
- 删除:
    -   与单链表不同,双向链表删除不需要保存上一个节点的信息,只需要获取到当前节点即可,因为存在前驱和后继节点
